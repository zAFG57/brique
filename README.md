**Jeu de briques !!**

>  Ce script python est un jeu de briques. Il a été crée en utilisant la librery py-game.
>  Je l'ai codé afin d'aider un camarade de classe qui l'avait pour sujet de dm
>  Ce jeu a obtenu la note de 20/20 en NSI. Il existe 3 niveaux avec un score qui se cumule au fur et à mesure des parties.
>  Il émet des sons à chaque contact avec des murs/briques/sol... De plus, une musique différente est diffusée selon si vous gagnez ou si vous perdez
